ImageIO介绍

Image IO允许应用程序读写大部分图片文件格式。起初是作为Core Graphics 框架的一部分，后来独立出来，允许开发者操作图像数据且非常高效。读取图片属性，提供颜色管理。（此接口osx v10.4 ios4.0）

Image I/O框架的特点：
1、mac平台最快的图像编码解码操作。
2、渐增的图片加载方式。
3、图片metadata信息的访问（ios不支持）。
4、有效的缓存。

lbs之两坐标点之间的距离

在LBS应用中，需要知到两坐标点之间的距离，从而在手机应用中实现离自己多少距离的附近的人员。

计算方法从Haversine formula 这个公式得来。
查看wiki : Haversine公式

下面写出的php实现，

public class LBS {
    private $earthRadius = 6367000;

    /*
     * Using the Haversine formula
     *
     *http://en.wikipedia.org/wiki/Haversine_formula
     * calculate the distance
     */
    public function getDistance($lant1,$lon1,$lant2,$lon2) {
        $nDlat = ($lant1 - $lant2) * pi() / 180;
        $nDlon = ($lon1 - $lon2) * pi() / 180;

        $nA = pow (sin($nDlat / 2),2) + cos($nDlat) * cos($nDlat) * pow(sin($nDlon / 2),2);

        $nC = 2 * atan(sqrt($nA),sqrt(1 - $nA));
        $nD = $earthRadius * $nC;
        return $nD;
    }
}

就可以计算地球上面两个位置点之间的距离。

使用他人app里面图片
##################

有时候我们看到某个app中有个有趣的图片，想获得这个图片，可以从该app的程序包中提取

1、首先在itunes里面打开ipa包。

2、使用归档工具解压查看。

3、有些png图片无法用”预览”程序显示,原因是Xcode在打包程序时，使用一个名为pngcrush的工具对png图片针对iphone进行了压缩优化处理，导致这些图片无法正常显示（可正常显示的png图片经pngcrush处理后无变化），pngcrush程序在Xcode的安装目录中存在，路径为/Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/usr/bin/pngcrush
使用方法为
$ /Applications/Xcode.app/Contents/Developer\
/Platforms/iPhoneOS.platform/Developer/usr/bin/pngcrush \
-revert-iphone-optimizations -q xx.png xx_revert.png

就可以将图片还原。
