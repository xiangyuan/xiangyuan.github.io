python打包ios ipa包

Sun 02 December 2012
By 向远

In python,ios.

我们知道在ios上生成的itunes包应该是一个ipa的后缀的包，而不是直接xcode生成的.app的包。

ipa包的组成
其中有两项：itunes 所认识的一个图片名为iTunesArtwork
第二个重要的是 .app的应用程序

其实生成ipa包很简单 就是通过 zip -r进行打包。

1、方法一将就.app包直接拖到itunes就可

但作为coder嘛 所以还是通过程序来吧。

下面以python还达到此目的，原理如上面所讲的。

class package:
    def __init__(self,origin_path,output_path):
        self.origin_path = origin_path
        self.output_path = output_path
        homeDirectory = os.environ['HOME']

        project_directory = homeDirectory + os.path.sep + origin_path
        self.project_directory = project_directory

    def get_current_path(self,path_name):
        '''Get the current path

        return path_name;
    def exec_cmd(self,cmd):
        retCode = -1
        try:
            print cmd

        except EnvironmentError:
            print 'command execute encounter some error'
        return retCode
    def get_app_name(self):
        if os.path.isdir(self.project_directory + os.path.sep + 'build/Release-iphoneos'):
